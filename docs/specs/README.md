# Tài liệu Dự án Tử Vi AI Web

Thư mục này chứa tài liệu kỹ thuật và đặc tả cho dự án Tử Vi AI Web.

## Cấu trúc Tài liệu

- `architecture.md` - Kiến trúc hệ thống và luồng dữ liệu
- `api-spec.md` - Đặc tả API và tRPC routers
- `database-schema.md` - Cấu trúc database và quan hệ
- `frontend-guide.md` - Hướng dẫn phát triển frontend
- `deployment.md` - Hướng dẫn triển khai và cấu hình
- `development-workflow.md` - Quy trình phát triển và coding standards
- `feature-specs/` - Đặc tả chi tiết từng tính năng

## Cập nhật Tài liệu

Tài liệu cần được cập nhật khi:
- Thêm/sửa API endpoints
- Thay đổi database schema
- Thêm tính năng mới
- Cập nhật quy trình deployment

## Liên hệ

Mọi thắc mắc về tài liệu, vui lòng tạo issue hoặc liên hệ team lead.
