# Kiro Steering Files

Thư mục này chứa các file steering để hướng dẫn Kiro làm việc hiệu quả với dự án Tử Vi AI Web.

## File Structure

- `product.md` - Tổng quan sản phẩm và tính năng
- `tech.md` - Tech stack và công cụ
- `structure.md` - Cấu trúc dự án và thư mục
- `rules.md` - Quy tắc development và code standards
- `context.md` - Context làm việc và guidelines
- `patterns.md` - Các pattern và ví dụ code thường dùng
- `priorities.md` - Ưu tiên và roadmap dự án

## Mục đích

Các file này giúp Kiro:
- Hiểu rõ context và mục tiêu của dự án
- Áp dụng đúng patterns và best practices
- Tạo code nhất quán với codebase hiện tại
- Ưu tiên đúng tính năng theo roadmap
- Tuân thủ coding standards và quy trình

## Cập nhật

Cần cập nhật steering files khi:
- Thay đổi tech stack hoặc architecture
- Thêm patterns mới hoặc best practices
- Cập nhật priorities và roadmap
- Thay đổi quy trình development

## Sử dụng

Kiro sẽ tự động đọc và áp dụng các hướng dẫn trong steering files khi làm việc với dự án.
