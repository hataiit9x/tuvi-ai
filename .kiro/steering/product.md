# Product Overview

Tử Vi AI Web is a Vietnamese fortune-telling and astrology platform that combines traditional Eastern metaphysical practices with AI-powered analysis.

## Core Features

- **Tử Vi (Purple Star Astrology)**: Generate and analyze Vietnamese astrology charts based on birth data
- **Numerology**: Calculate life path, soul, personality, and destiny numbers using Pythagorean system
- **Zodiac Forecasts**: Chinese zodiac predictions and compatibility analysis
- **Auspicious Dates**: Find favorable dates for weddings, business openings, travel, etc.
- **Tết Tools**: Lunar New Year utilities including Xông Đất (first visitor), lucky colors, and lucky money suggestions
- **Compatibility Analysis**: Compare two people's astrological and numerological profiles

## Target Audience

Vietnamese-speaking users interested in traditional fortune-telling, astrology, and feng shui guidance.

## Key Characteristics

- All user-facing content is in Vietnamese
- AI analysis uses LLM integration for personalized interpretations
- Authenticated users can save reading history
- Admin panel for LLM configuration management
